1. The Strategy in One Page

You don't have investors. That's fine. What you have is enough.

The play is simple: build the entire product using free tiers and open-source models, charge ₹499 per student per month, and use that revenue to graduate to paid infrastructure as you grow. No fund-raising, no dilution, no pressure.

5. 12-Week Build Plan (Solo, Weekends + Evenings)

You're one person. This plan assumes ~15 hours/week. Each week ends with a working, demoable milestone.

Week 1 - Accounts & Foundation

Sign up: GitHub, Vercel, Oracle Cloud, Supabase, Cloudflare, Groq, Razorpay (test mode), Meta WhatsApp Cloud, Sentry, UptimeRobot. Spin up an Oracle Always-Free ARM VM (Ubuntu 22, 4 cores / 24 GB). Create a GitHub repo ananta. Set up base Next.js + NestJS skeleton. Deploy "Hello Ananta" page to Vercel.

Week 2 - Auth & Onboarding

Use Supabase Auth with phone OTP (free tier). Build signup flow: phone → OTP → collect name, grade (10/11/12), medium (EN/KN), school, parent phone. Generate parent-link code. Build a simple Parent app screen that takes the code and links accounts.

Week 3 - Curriculum Ingestion (Math, Grade 10)

Download KSEEB Class-10 Maths textbook PDF. Use a script to OCR and chunk into chapters/topics. Use Groq (free Llama-3-70B) to generate a curriculum JSON. Store in Supabase. Build admin page to review/edit topics.

Week 4 - First AI Lesson (English-medium)

Pick one Math topic. Use Groq to generate the lesson script. Use Coqui XTTS to generate English voice-over. Use Manim to render 3-4 animations. Composite with ffmpeg into one MP4. Upload to R2. Manually review. This is your proof of concept.

Week 5 - Kannada Track for the Same Lesson

Run the same script through a Kannada adaptation prompt. Use AI4Bharat Indic-Parler-TTS for the Kannada voice. Re-composite with the same animations. Now you have two parallel lessons from one pipeline. Validate that both feel native, not translated.

Week 6 - Lesson Player & VOD

Build the student lesson page. HLS player (Video.js or Shaka). Lessons stream from R2 through Cloudflare CDN. Implement heartbeat-every-30s pinging your backend. Build basic attendance computation.

Week 7 - Live Class MVP

Install LiveKit OSS on Oracle VM. Build a live room: AI teacher avatar plays the pre-rendered video at 7 PM; students join; chat sidebar. No live Q&A yet. Test with yourself + 1 friend. Recording = re-upload the source MP4 to R2.

Week 8 - Live Q&A Bot

Students post doubts in chat (text only at v0). A simple LLM call to Groq answers with RAG over the lesson transcript. Answer appears as a chat reply + an optional TTS voice clip. Skip avatar for v0 - voice + text is enough.

Week 9 - Parent Notifications

Set up Meta WhatsApp Cloud API. Create a missed-class template, get it approved (~24 hrs). Wire attendance pipeline: if watched_pct < 50%, send WhatsApp to parent. Test end to end with your own number.

Week 10 - Weekly Test

Build the assessment service: pull last 5 lessons, ask Groq for 30 questions (mix of MCQs and fill-blanks). Build a timed test player with tab-switch detection. MCQ auto-grade instantly; subjective LLM-graded async. Parent gets WhatsApp with the score.

Week 11 - Payments & ₹499 Subscription

Integrate Razorpay. Set up the ₹499/month plan with UPI Recurring Mandate. Build pricing page, signup-with-trial flow, billing dashboard. Activate live mode (needs PAN + bank account KYC).

Week 12 - Polish + First 5 Students

Fix bugs. Write a landing page. Make a 90-second demo video. Post in 3 Karnataka student WhatsApp groups + ask 5 friends with school-going kids to try a 1-week trial. Start the first paid Maths week. Launch.

8. The Hard Rules (To Actually Stay Free)

  • Render all lessons offline, never live. The 7-8 PM slot only serves pre-rendered video. Live AI is only for the Q&A overlay.
  • Delete recordings older than 2 months until you outgrow R2's 10 GB. Compress at H.265 (50% smaller).
  • Cap concurrent viewers at 50 per class in v0. If demand exceeds that, you need a paid VPS - which means you're earning enough to afford it.
  • Run the 1-hour live class on Oracle VM, not your laptop. Your laptop can die mid-class; the Oracle VM won't.
  • Don't sign up for paid plans of free services. You'll do it accidentally. Stay disciplined - Supabase Pro, Vercel Pro, etc. are tempting but unnecessary for the first 50 students.
  • Use WhatsApp templates carefully. Free tier is 1,000 service conversations/month. Batch and dedupe before sending.
  • Keep one credit card for emergencies only. If a free tier suddenly bills you, you want to catch it fast.

10. What to Tell Students & Parents

"Ananta is an AI-teacher tutoring platform built for Karnataka SSLC and PU students. Live class daily at 7 PM, recorded so you can watch later, weekly tests every Saturday, parents get a WhatsApp report. English-medium and Kannada-medium tracks. ₹499 a month. Try it free for 7 days."

Be upfront that you're a one-person team in v0. Parents value honesty more than they value polish. Many will root for you.

11. Two Things Not To Do

  1. Don't build all 5 subjects before launching. Launch with Maths Grade 10 only. Validate that students pay ₹499 for it. Then expand.
  2. Don't worry about scale before you have 10 students. The free stack will handle 50 students comfortably. Premature optimization will burn weeks you don't have.
